# Raw Nutrition - Location Finder
---

The application is built around a professional index page for a fictional company, RAW Nutrition. Its primary function is to accurately display the user's current device location on a dynamic map.
---

### Key Features & Requirements Met

- **Fictional Company Page**: Developed a professional landing page for "RAW Nutrition".
- **Geolocation Integration**: Uses the browser's Geolocation API to retrieve the device's latitude and longitude.
- **Callback Functions**: Implements callback functions (successLocation and errorLocation) to handle the asynchronous location request.
- **Map Initialization**: The Mapbox is only initialized upon user interaction (by clicking the "HERE" button).
- **Marker Display**: A custom colored mapbox marker, to be placed on the exact location of the user.
